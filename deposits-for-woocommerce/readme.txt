WooCommerce Deposits & Partial Payments allows customers to pay for products using a partial payment.

== Description ==

WooCommerce Deposits plugin enables customers to conveniently make payments for their purchased products using partial payments or deposits. With the versatile partial payment WooCommerce plugin, you can effortlessly manage partial payments, require a deposit, or provide customers with the option to pay a percentage or a fixed price upfront.
